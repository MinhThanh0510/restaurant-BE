# L'Aura Restaurant — Backend API

RESTful API for a Restaurant Table Reservation & Menu Management System.

## Tech Stack
- Node.js, Express.js (MVC Architecture)
- MongoDB Atlas (Mongoose)
- JWT Authentication, bcryptjs, Nodemailer
- Socket.io, PayPal REST API

## Features
- RBAC (Customer / Admin roles)
- Reservation workflow with real-time conflict detection
- PayPal payment integration with webhook auto-confirmation
- Inventory tracking with Low Stock alerts
- Real-time Live Chat via Socket.io

## Getting Started
# Install dependencies
npm install

# Run development server
npm run dev
